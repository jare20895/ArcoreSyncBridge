# User Persona Maps

## Persona: Data Steward
- Role: Owns data quality and mappings
- Goals: Reliable syncs, minimal drift, clear audit trails
- Pain points: Manual reconciliation, unclear errors

## Persona: Integration Engineer
- Role: Builds and operates integrations
- Goals: Stable APIs, reproducible config, observability
- Pain points: Throttling, schema drift, brittle scripts

## Persona: Business Analyst
- Role: Consumes SharePoint lists
- Goals: Up-to-date lists without manual exports
- Pain points: Stale data, missing fields
