# Getting Started Guide

## Prerequisites
- Access to Arcore SyncBridge console
- A Postgres connection string visible to the platform
- A SharePoint site with list creation permissions

## Steps
1. Create a Postgres connection profile.
2. Create a SharePoint connection profile.
3. Define a sync definition for the source table.
4. Review field mappings and sharding rules.
5. Run provisioning to create the SharePoint list.
6. Trigger the first sync run.
