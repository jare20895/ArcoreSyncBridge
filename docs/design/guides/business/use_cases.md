# Use Cases

## Use case 1: Project portfolio visibility
**Actor:** PMO analyst
**Scenario:** Sync project status from Postgres into SharePoint lists segmented by status.
**Outcome:** Stakeholders consume up-to-date project data in Teams.

## Use case 2: Compliance evidence tracking
**Actor:** Compliance manager
**Scenario:** Sync audit records into SharePoint lists with controlled access.
**Outcome:** Auditors access curated records without direct DB access.

## Use case 3: Asset inventory
**Actor:** Operations lead
**Scenario:** Sync asset inventory into SharePoint for field updates.
**Outcome:** Field updates are written back to Postgres (Phase 3).
